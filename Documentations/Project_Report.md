# SQL Data Cleaning Project Report

## Objective

To clean and standardize the World Layoffs dataset using SQL so it can be used for accurate analysis.

## Challenges

- Duplicate records
- Blank strings
- NULL values
- Incorrect date datatype
- Inconsistent text formatting


## Solution

The dataset was cleaned using SQL by:

- Creating a staging table
- Removing duplicate records
- Standardizing text values
- Handling missing values
- Converting blank strings into NULL
- Formatting date columns
- Removing unnecessary records

## Results

The final dataset is:

- Free from duplicate records
- Standardized
- Suitable for analysis
- Better structured for reporting

## Key Learnings

Through this project I learned:

- Professional SQL workflow
- Importance of preserving raw data
- Data standardization
- Handling missing values correctly
- Self Join
- Window Functions
- Business reasoning behind data cleaning decisions
